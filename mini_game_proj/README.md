# 나폴레옹 전쟁 전략 게임

유럽 전략 지도에서 프랑스 제국(나폴레옹)을 맡아 군대를 이동시키고 도시를
점령하는 턴제 전략 게임. 현재는 **전략 지도 MVP** 단계다.

## 실행

```bash
# 프로젝트 루트(claude_study_Kor)에서
.venv/Scripts/python.exe mini_game_proj/main.py
```

pygame이 없으면: `uv pip install pygame`

## 조작

| 입력 | 동작 |
|------|------|
| 좌클릭 (아군 부대) | 부대 선택 |
| 좌클릭 (초록 테두리 도시) | 선택한 부대를 인접 도시로 이동 |
| Space / Enter | 턴 종료 (이후 AI 진영 행동) |
| Esc | 종료 |

- 빈 도시로 이동하면 **점령**, 적 부대가 있으면 **전투**가 벌어진다.
- 부대 아이콘의 노란 점 = 장군(근위대) 동반. 전투력 +15%.
- 주요 도심(파리/런던/모스크바 등)은 방어 시 +25% 보너스.
- 모든 적 군사 진영을 제압하면 **정복 승리**.

## 육상 전투맵 (전술 전투)

플레이어가 관여하는 전투(공격/방어)는 **전술 전투맵**으로 전환된다. AI끼리의
전투는 전략 판정으로 즉시 처리된다.

- 군대는 여러 **부대(연대)** 로 분할: 보병/기병/포병/근위(장군 동반 시).
- 지형: 평원·설원·산맥·강·다리·도심. 이동 비용과 공·방 보정이 다르다.
  강은 다리로만 건널 수 있고, 산맥·도심은 방어 보너스가 크다.
- 전장 종류는 위치로 결정: 주요 도심→도심, 러시아권→설원, 알프스·발칸→산맥, 그 외→평원.

### 전투 조작

| 입력 | 동작 |
|------|------|
| 좌클릭 (아군 부대) | 선택 |
| 드래그 (박스) | 여러 부대 **묶어서 선택** |
| Shift+클릭 | 추가/해제 선택 |
| 좌클릭 (빈 칸) | 선택 부대 이동 |
| 좌클릭 (적 부대) | 사거리 내 공격(필요 시 접근 후) |
| 우클릭 | 선택 해제 |
| Space | 턴(페이즈) 종료 |
| A | 자동 전투(즉시 판정) |

공격군이 도심 타일을 점령하거나 한쪽이 전멸하면 전투 종료. 생존 병력은 전략
지도의 해당 군대 병력으로 반영된다.

## 해전 (해상 전술 전투)

바다 노드(대서양·북해·발트해·서지중해·동지중해·흑해)를 항로로 이동하는
**함대**끼리 충돌하면 해전 전술맵으로 전환된다. 영국이 최강 해군(40k)이다.

- 함대는 **전열함·프리깃·브릭·슬루프**로 나뉜다. 전열함은 강하지만 느리고,
  슬루프는 약하지만 빠르다.
- **현측 포격**(broadside, 원거리)으로 싸우며 인접 시 양측이 서로 포격한다.
- **바람(풍향)** 이 이동에 영향: 순풍이면 빠르고 역풍이면 느리다(우측 패널/상단 표시).
- 외해·천해·해안(섬)으로 구성되며 해안은 통과 불가, 천해는 느리다.

조작은 육상전과 동일(좌클릭 선택/이동/포격, 드래그 묶음선택, Space 종료, A 자동).

## 에셋 / 효과음

외부 파일 의존 없이 런타임에 생성한다.

- **스프라이트**([sprites.py](game/sprites.py)): 병사 4종(보·기·포·근위)과 함선
  4종을 `pygame.draw`로 그려 진영색을 입힌다. 전략 지도·전투맵 모두에서 사용.
- **효과음**([audio.py](game/audio.py)): 대포·머스킷·이동·돛·선택·승리·패배를
  표준 라이브러리로 합성. 오디오 장치가 없으면 자동으로 무음 처리.
- 국기(`assets/flags/`)와 유럽 지도(`assets/europe_map.png`)만 위키미디어에서
  내려받은 이미지 파일이다.

## 구조

```
mini_game_proj/
├── main.py            # 진입점 + 전략 루프 + 전투 디스패처(육상/해전 분기)
└── game/
    ├── settings.py    # 화면/색상/파라미터
    ├── map_data.py    # 유럽 노드 그래프(육지+바다), 진영, 초기 군대/함대
    ├── army.py        # 군대/함대 모델 (domain: land/sea)
    ├── battle.py      # 전략 즉시 전투 판정 (AI vs AI)
    ├── tactical.py    # 육상 전투맵: 지형·부대·전투·AI·렌더
    ├── naval.py       # 해전 전투맵: 바다·바람·함선·포격·AI·렌더
    ├── sprites.py     # 병사/함선 스프라이트 생성
    ├── audio.py       # 효과음 합성
    ├── game_state.py  # 월드 상태, 턴 진행, 이동/점령/승리
    ├── ai.py          # AI 진영 휴리스틱(육·해 공용)
    └── ui.py          # 전략 지도 렌더링 + 폰트 + 국기/지도 로드
```
