# 나폴레옹 전쟁 전략 게임
- 19세기 유럽 나폴레옹 전쟁을 배경으로 함
- 유럽 지도에서 외교, 정치, 군사 사용가능
- 육상전은 개인 유닛을 컨트롤하는 것이 아니라 여러 유닛을 묶어서 쓸 수 있음.

## 사용 스택
- Pygame
- Python 3.13

## 개발 명령
- uv venv로 생성된 .venv(메인)에 있는걸 활용하여 패키지 설치하고 진행


## RULES
- 게임 진행 방식은 군사 승리, 진영 승리, 정복 승리가 있음.
- 지도에서는 큰 Unit과 같은 것들이 그 진영에 병력에 누구 있고, 장군 근위대가 있음을 보여줌
- 서로 부딪혔을 때 전투맵으로 넘어감

## 전투맵 Rules
- 전투맵은 강, 다리, 평원, 설원, 산맥, 그리고 도심임.
- 파리, 런던, 모스크바, 이스탄불, 마그리드, 로마, 빈, 베를린과 같은 주요 도심들은 도심 구현 필요

## 해전
- 바다에서는 전열함, 프리깃, 브릭, 슬루프 선과 같은 해전 구현 필요
